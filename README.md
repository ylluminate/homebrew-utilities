# Homebrew Tap

Custom [Homebrew](https://brew.sh) formulas.

## Usage

```bash
brew tap ylluminate/utilities
```

## Formulas

| Formula | Description |
|---------|-------------|
| [plocate](https://github.com/ylluminate/plocate-macos) | Fast file search for macOS using posting lists and trigram indexing |

### plocate

```bash
brew install --HEAD ylluminate/utilities/plocate
sudo updatedb
```

See the [plocate-macos README](https://github.com/ylluminate/plocate-macos) for configuration and usage.
